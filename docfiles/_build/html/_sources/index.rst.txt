Welcome to *autoBOT*'s documentation
===================================

`autoBOT <https://github.com/SkBlaz/autobot/>`_ is an autoML system for automatic explanable classification of *texts*. It includes automatically generated features at different granualrities, evolved to offer optimal performance/explainability tradeoff.
